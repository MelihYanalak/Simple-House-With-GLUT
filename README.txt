In order to run the project in Linux you could simply use the make command bu using terminal in the project directory.
Or manually you can run the commands respectively:
	 gcc -g -Wall -o out 3d_house.c -lglut -lGL -lGLU
	./out

While the program is running, you can use w,a,s,d,k,m buttons to change the angle and position of the camera.
